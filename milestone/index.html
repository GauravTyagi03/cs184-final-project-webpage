What We Accomplished:

In the first week, all team member worked independently to better understand OptiX, looking at tutorials and the OptiX library. One valuable resource was an existing tutorial on raytracing with OptiX, which we used as a starting point to initialize OptiX. The second week we created our repo based on a tutorial repo we found online and modified it to incorporate with the homework 3 codebase.

Building the project and getting the GPU to render images was the most challenging part of the checkpoint. When incorporating OptiX into the codebase, we ran into several key challenges.

  • Configuring CMake to find OptiX and CUDA while adhering to the structure from homework 3
  • The vector library from homework was unusable, so we instead switched to using the C++’s uint2 struct.
